Movie API exercise with NestJS to build enterprise NodeJS applications

🎞 Get information of a movie / movies
✍️ Create movie info
❌ Delete movie info
🔄 Update movie info

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
